# Fundamentos de Programación

Este repositorio contiene ejercicios de fundamentos de programación DAM organizados por temas.

## Estructura del Proyecto

- **PRÁCTICA 1.1 ALGORITMOS CON INSTRUCCIONES ALTERNATIVAS**: Contiene ejercicios que utilizan estructuras de control alternativas


- **2_2 ALGORITMOS SECUENCIALES PRÁCTICA**: Contiene ejercicios de algoritmos secuenciales.


- **3_2ALGORITMOS CON ESTRUCTURAS ALTERNATIVAS Y REPETITIVAS**: Contiene ejercicios con estructuras alternativas y repetitivas.


- **EJERCICIOS CON VECTORES I**: Contiene ejercicios relacionados con vectores, incluyendo:
    - `ejercicio_1.py`: Generación de boletos de Bonoloto.
    - `ejercicio_2.py`: Simulación de una máquina expendedora.


- **EJERCICIOS CON VECTORES II**: Ejercicios con vectores.
